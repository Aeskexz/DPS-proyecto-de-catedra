Este proyecto está licenciado bajo la **Licencia Creative Commons Atribución-CompartirIgual 4.0 Internacional (CC-BY-SA 4.0)**.

### ¿Qué significa esto?

**CC-BY-SA 4.0** es una licencia que permite:

 **Uso Comercial** - Puedes usar el material con fines comerciales  
 **Modificación** - Puedes modificar, adaptar y mejorar el material  
 **Distribución** - Puedes distribucjir el material original o modificado  
 **Uso Privado** - Puedes usar el material para uso privado  

### Requisitos

Cuando uses, modifiques o distribuyas este proyecto, **DEBES**:

1. **Atribución (BY)** - Brindelar crédito apropiado al proyecto y sus autores
   - Menciona el nombre original del proyecto
   - Incluye un enlace a la licencia
   - Indica si se realizaron cambios

2. **CompartirIgual (SA)** - Distribuir trabajos derivados bajo la **misma licencia** (CC-BY-SA 4.0)
   - Si modificas el proyecto, debes compartir con la misma licencia
   - Esto asegura que las mejoras también beneficien a la comunidad

### Prohibiciones

 Los siguientes usos **NO están permitidos** sin permiso:
- Reivindicar que patentes, marcas registradas, o derechos de autor son tuyos
- Usar medidas tecnológicas para restringir el acceso al material
- Violar derechos de privacidad o difamación

##  Más Información sobre Creative Commons

### ¿Qué es Creative Commons?

Creative Commons es una organización sin fines de lucro que proporciona **seis licencias de derechos de autor** gratuitas que ofrecen una manera estándar de ceder derechos de autor en un trabajo creativo.

### Licencias CC Disponibles

| Licencia | Símbolo | Descripción |
|----------|---------|-------------|
| **CC-BY** | BY | Solo requiere atribución |
| **CC-BY-SA** | BY-SA | Atribución + Compartir con la misma licencia |
| **CC-BY-NC** | BY-NC | Atribución + Sin uso comercial |
| **CC-BY-NC-SA** | BY-NC-SA | Atribución + Sin comercial + Compartir igual |
| **CC-BY-ND** | BY-ND | Atribución + Sin derivados |
| **CC-BY-NC-ND** | BY-NC-ND | Atribución + Sin comercial + Sin derivados |

### ¿Por qué CC-BY-SA 4.0?

Se eligió **CC-BY-SA 4.0** para este proyecto porque:

1. **Promueve el código abierto** - Cualquiera puede mejorar el proyecto
2. **Beneficia a la comunidad** - Las mejoras vuelven al proyecto
3. **Transparencia legal** - Términos claros y justos
4. **Reconocimiento** - Los contribuyentes son reconocidos
5. **Sostenibilidad** - Asegura que el conocimiento siga siendo compartido

##  Cómo Implementar CC-BY-SA 4.0 en tu Proyecto

### 1. Agregar Aviso en tu Código

Al inicio de los archivos de código, puedes agregar:

```javascript
/**
 * Sistema de Citas Médicas
 * Licensed under CC-BY-SA 4.0
 * https://creativecommons.org/licenses/by-sa/4.0/
 */
```

### 2. Agregar Aviso en tu README

```markdown
##  Licencia

Este proyecto está bajo la licencia **Creative Commons Atribución-CompartirIgual 4.0 Internacional**

-  [LICENSE](LICENSE) - Texto completo en inglés
-  [LICENSE.es](LICENSE.es) - Texto completo en español

Bajo esta licencia puedes:
-  Usar comercialmente
-  Modificar
-  Distribuir

Con la condición de:
-  Mantener atribución
-  Compartir derivados bajo CC-BY-SA 4.0

Más información: https://creativecommons.org/licenses/by-sa/4.0/
```

### 3. Agregar Badge en tu README

```markdown
[![CC BY-SA 4.0][cc-by-sa-shield]][cc-by-sa]

[cc-by-sa]: http://creativecommons.org/licenses/by-sa/4.0/
[cc-by-sa-image]: https://licensebuttons.net/l/by-sa/4.0/88x31.png
[cc-by-sa-shield]: https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg
```

### 4. En package.json

